# ECON-690-Project

This study uses data from the 2022 Survey of Consumer Finances (SCF), a nationally representative triennial
survey conducted by the Federal Reserve Board that provides detailed information on U.S. households’
balance sheets, income, credit access, and financial behaviors. The SCF employs a dual-frame sampling
design that oversamples wealthy households and uses survey weights to ensure national representativeness.
To address item nonresponse, the SCF uses multiple imputation, with each household appearing in five
implicates. All analyses account for the survey design by incorporating household-level weights.
The dataset includes information on Buy Now, Pay Later (BNPL) usage as well as a rich set of demographic,
financial, and behavioral variables, enabling a comprehensive examination of the relationship between BNPL
adoption and household debt, liquidity, and financial stress outcomes.
